# PrivacyShield — DPDP Compliance Platform

DPDP Act 2023 compliance assessment and implementation planning tool for Indian banks and NBFCs.

## Quick Start

```bash
npm install
npm run dev
```

Opens at http://localhost:5173

## Features

- **DPDP Act Knowledge Base** — Complete Act breakdown with banking-specific interpretation
- **Interactive Gap Assessment** — 43 controls scored by your actual answers
- **Compliance Report** — Auto-generated score with area breakdown and risk exposure
- **Implementation Roadmap** — Jira-ready Epics, Stories, Acceptance Criteria, Test Cases
- **AI DPDP Advisor** — Ask compliance questions (requires Anthropic API access)
